HTML Tables

HTML tables are elements within web design that allow web developers to organize data such as text, images, and links in rows and columns.

Explanation about the tags in the table:

Tags Description
<table> Declare a table into HTML
<tr> Declare a row in a table
<th> Declare a table column title
<thead> Declare the row header
<td> Declare a cell/data in the table

Here is the code ðŸ˜‰:

  

    <div class="flip-card" tabIndex="0">
      <div class="flip-card-inner">
        <div class="flip-card-front">
          <h3>Hover, please!</h3>
        </div>
        <div class="flip-card-back">
          <h3>Whoaaa!!!</h3>
        </div>
      </div>
    </div>


    .flip-card {
      background-color: transparent;
      width: 300px;
      height: 300px;
      perspective: 1000px;
    }
    
    .flip-card-inner {
      position: relative;
      width: 100%;
      height: 100%;
      text-align: center;
      transition: transform 0.6s;
      transform-style: preserve-3d;
      box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.2);
      backface-visibility: hidden;
      -moz-backface-visibility: hidden;
    }
    
    .flip-card:focus {
        outline: 0;
    }
    
    .flip-card:hover .flip-card-inner,
    .flip-card:focus .flip-card-inner{
      transform: rotateY(180deg);
    }
    
    .flip-card-front,
    .flip-card-back {
      position: absolute;
      width: 100%;
      height: 100%;
    }
    
    .flip-card-front {
      background: linear-gradient(to left, #4364f7, #6fb1fc);
      color: black;
      z-index: 2;
      display: flex;
      justify-content: center;
      align-items: center;
    }
    
    .flip-card-back {
      background: linear-gradient(to right, #4364f7, #6fb1fc);
      color: white;
      transform: rotateY(180deg);
      z-index: 1;
      display: flex;
      justify-content: center;
